Programa feito utilizando o Framework QT Creator.
Necessário a intalação do curl.
No codigo é utilizado um banco de dados chamado "profdb.db", para alterar deve mudar o nome e o endereço do banco de dados no arquivo main.cpp
O objetivo do programa é controlar os emprestimos de materiais do almoxarifado ao alunos,é um formulario onde são colocados os dados do aluno e os materiais que serão emprestados para projeto, salvando esses dados em um banco de dados SQL e enviando um email ao professor responsavel pelo projeto avisando sobre o aluno e os materiais que foram solicitados.

Program made using the QT Creator Framework.
Curl installation required.
The code uses a database called "profdb.db", to change it you must change the name and address of the database in the main.cpp file
The objective of the program is to control the loan of materials from the warehouse to students, it is a form where the student's data and the materials that will be loaned for the project are placed, saving this data in a SQL database and sending an email to the responsible teacher by the project informing about the student and the materials that were requested.
