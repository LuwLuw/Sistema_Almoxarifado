This folder contains Aspose.Email for C++ 22.7 release.

Suffix _vc14x86  of the file names means that library requires Microsoft Visual Studio 2017 in x86 mode.
Suffix _vc14x64 in file names means that library requires Microsoft Visual Studio 2017 in x64 mode.

Folder "lib" contains libraries used by Aspose.Email for C++.
Folder "include" contains all neccessary header files.
Folder "example" contains Visial Studio 2017 example shows how to use Aspose.Email for C++ library.
